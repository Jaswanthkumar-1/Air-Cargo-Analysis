# ✈️ Air Cargo Analysis — SQL Course-End Project

## Overview
SQL-based analysis for Air Cargo aviation company covering
customer behaviour, route analysis, and ticket sales.

## Tools Used
- MySQL
- SQL (DDL, DML, Views, Stored Procedures, Functions, Cursors)

## Dataset
- 50 Customers | 50 Passengers | 49 Routes | 50 Tickets

## Queries Covered
20 SQL queries including:
- Joins, Group By, Having
- Window Functions (PARTITION BY)
- Stored Procedures & Functions
- Cursors, Views, Rollup, Index

## Key Findings
- Total Revenue: $15,369 ✅ Crossed $10,000 target
- Busiest Route: JFK → LAX (Route 4)
- Top Brand: Emirates (14 customers)
- Business Class Revenue: $6,034
